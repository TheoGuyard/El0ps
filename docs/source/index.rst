=========
El0ps
=========
*— An Exact L0-Problem Solver —*

--------


``el0ps`` is a Python package allowing to solve L0-penalized problems.
It is designed to be numerically efficient and supports a wide range of loss and penalty functions.
You can pick from ``el0ps``'s already-made estimators or **customize your own** by combining the available datafits and penalties.


Installing ``el0ps``
--------------------

``el0ps`` is available on PyPi. Get the latest version of the package by running

.. code-block:: shell

   $ pip install el0ps
